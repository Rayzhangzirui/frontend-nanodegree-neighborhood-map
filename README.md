# Neighborhood Map Project

This project use Google Maps API and Flickr API to display some main attractions in New York City.

### Using the app

* Open the ```index.html``` file in a modern browser.
* Click the icon on the top left to have the list view.
* Click markers or list items to select a location to open the infoWindow.
* Click left or right in the infoWindo to navigate the carousel.
* Type into the filter box to filter the shown locations.
